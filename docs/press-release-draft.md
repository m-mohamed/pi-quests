# Press Release Draft

Quest is a Pi-native orchestration package for long-running autonomous coding.

Pi gives the primitive runtime. Quest adds proposal-first and validation-first execution. Trials add evals-and-traces optimization around that runtime.

This release introduces the first benchmark substrate for Quest:

- `quest-headless` for machine-driven benchmark runs
- Harbor integration for official Terminal-Bench runs
- a local smoke adapter plus upstream-runner overlay for SlopCodeBench
- benchmark provenance on traces and replay datasets

The release is methodology-first. Quest is public, the adapters are public, and the reproducibility steps are public. Score claims are published only after stable reruns exist on real benchmark tasks.
